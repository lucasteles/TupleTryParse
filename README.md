# TupleTryParse
Extensions for use TryParse of types with a tuple result

```cs
var value = "123";
var (success, parsedValue) = Parser.TryParseInt(value);
```

## Installing
Install from Nuget

```powershell
Install-Package TuppleTryParse 
```

Install from dotnet core CLI

```powershell
dotnet add package TuppleTryParse 
```
